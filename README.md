# Hide Stats

A Spicetify extension that removes listening statistics from Spotify for a cleaner, distraction-free experience.

## What it hides
- Monthly listener count on artist pages
- Play count on songs and albums
- Follower count, monthly listeners and city stats on artist About page

## Install
1. Open Spotify with Spicetify installed
2. Go to the Marketplace
3. Search for "Hide Stats"
4. Click Install

## Manual Install
1. Download `hide-stats.js`
2. Copy it to `%APPDATA%\spicetify\Extensions\`
3. Run `spicetify config extensions hide-stats.js`
4. Run `spicetify apply`
